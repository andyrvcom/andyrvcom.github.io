FAQ

Questions before you export?

Short answers for practical QR code workflows.

Which formats can I export?

SVG, PNG, and JPG. SVG stays vector-perfect, while PNG and JPG are generated at high resolution.

Can I use it for print?

Yes. Use SVG for design tools or a large PNG/JPG when your print workflow requires raster images.

Is it free to use?

This static page generates codes in the browser. There is no signup flow or server upload.

What can I encode?

Links, plain text, mailto emails, phone calls, Wi-Fi credentials, and any custom string.
